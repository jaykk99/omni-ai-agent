Monico Agent

Local Installation

Mac

pip install -r requirements.txt
uvicorn app:app --reload

Linux

pip install -r requirements.txt
uvicorn app:app --reload

Windows

pip install -r requirements.txt
uvicorn app:app --reload

Integrations

  • Telegram: python agent.py --telegram
  • Slack: python agent.py --slack
  • SMS: python agent.py --sms
